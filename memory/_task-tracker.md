## Format

```
### [ID] [STATUT] — [TITRE]
**Agent** : [Agent responsable]
**Début** : [Date]
**Fin** : [Date ou EN COURS]
**Description** : [Ce qui doit être fait]
**Progression** : [Pourcentage ou étape]
**Blocages** : [Problèmes rencontrés]
```

Statuts : 🔄 IN_PROGRESS | ✅ DONE | ⏸️ PAUSED | 🚧 BLOCKED
