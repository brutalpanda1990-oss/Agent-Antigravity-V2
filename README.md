## 🧠 C'est quoi ce système ?

Un système d'équipe IA complète organisé en **5 niveaux hiérarchiques**, activant automatiquement le bon expert selon ta demande. Il n'est pas lié à un projet spécifique — il est **global et réutilisable** dans toutes tes applications.

## 📦 Architecture

## ⚡ Démarrage rapide

1. **Ouvre Antigravity** dans ton projet
2. **Tape** ce que tu veux faire en langage naturel
3. Le système **route automatiquement** vers le bon agent
4. Utilise `/clarify` si ta demande est complexe
5. Utilise `/bootstrap` pour initialiser la mémoire projet

## 🔑 Commandes principales

| Commande | Action |
|---|---|
| `/clarify [demande]` | Structurer une demande complexe |
| `/plan-sprint [feature]` | Planifier un sprint |
| `/debug-issue [problème]` | Debug systématique |
| `/code-review` | Revue de code complète |
| `/security-scan` | Audit sécurité OWASP |
| `/bootstrap` | Scanner le projet actif |
| `/status` | Voir l'état des tâches |
| `/skill-import [source]` | Importer un skill externe |
